# 变更日志

## 2026-04-30

### 修复

- 修复用户无法在个人主页查看自己用户 ID 的问题。
- 在个人主页用户信息区域新增 `用户ID：{id}` 展示，并支持点击复制，便于好友添加、见证人邀请等需要填写用户 ID 的场景。
- 修复好友申请方向错误：发起人只能看到“已发送/等待对方接受”，接收人才会看到“收到的好友请求”并执行接受操作。
- 后端 `/friends` 现在同时返回当前用户发出的好友关系和收到的待处理申请，并通过 `direction` 字段区分 `outgoing` 与 `incoming`。

### 新增

- 新增每日断签检测任务：服务运行后每小时检查一次时间窗口，在每日 01:00 对前一天未打卡且未使用连击保护的参与者执行连击归零、信用分扣减和站内通知。
- 新增 Redis 周排行榜缓存：`/leaderboard/weekly` 会刷新 Redis Sorted Set 并按 XP 返回榜单；Redis 不可用时自动回退到数据库排序。
- 新增守约证书 PNG 图片生成：挑战完成时生成证书记录并写入 `image_url`。
- 新增后端规则单元测试，覆盖 XP 加成、等级计算、信用分边界和进度计算。
- 新增 Docker 前端镜像构建、Nginx SPA 托管和 `/api` 代理。
- 新增课程测试报告文档：`documents/TestReport.md`。

### 验证

- 已通过：`go test ./...`
